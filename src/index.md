# Book of USD (USD 實戰隨身指南)

**[通用場景描述 (Universal Scene Description, USD)](https://graphics.pixar.com/usd/release/index.html)** 是一種由 [皮克斯動畫工作室 (Pixar Animation Studios)](https://www.pixar.com/) 所開發的革命性框架，專為定義並處理複雜的 3D 場景資料而生。

本書旨在作為皮克斯[官方文件](https://graphics.pixar.com/usd/release/index.html)的最佳輔助伴讀指南。
